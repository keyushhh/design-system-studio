# Product Roadmap

This roadmap outlines planned enhancements for **Design System Studio**.

---

## Short-Term Goals (v1.1 - v1.2)

- **Enhanced Accessibility (a11y)**:
  - Add comprehensive ARIA attribute coverage across complex controls (`Select`, `Tabs`, `UploadZone`).
  - Conduct color contrast checks across custom theme variations.

- **Design Token Export Utilities**:
  - Add single-click token export options (Style Dictionary format, Tailwind config extensions, W3C Design Tokens standard format).

---

## Medium-Term Goals (v1.3 - v1.5)

- **Web Components Wrapper**:
  - Package core UI primitives as framework-agnostic Custom Elements for use in plain JS, Vue, Angular, or Svelte projects.

- **Theme Builder & Customizer**:
  - Add an in-browser theme builder allowing users to tweak brand palettes, corner radii, and font pairings with live preview and CSS download.

---

## Long-Term Goals (v2.0+)

- **Figma Token Synchronization**:
  - Provide GitHub Actions workflows to auto-sync design tokens with Figma Variables via REST API.

- **Expanded Slide Template Library**:
  - Introduce 10 additional presentation templates (e.g. Gantt charts, architectural topology diagrams, financial comparison grids).
