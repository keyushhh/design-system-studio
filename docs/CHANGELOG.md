# CHANGELOG

All notable changes to **Design System Studio** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-07-22

### Added
- Complete brand-agnostic design token system (`tokens/colors.css`, `tokens/typography.css`, `tokens/spacing.css`, `tokens/fonts.css`).
- Scoped Dark Theme override layer (`tokens/colors-dark.css`).
- Modular UI component library across Core, Forms, Navigation, Feedback, and Presentation categories.
- 14 Master Presentation slide frame renderers for executive slide deck creation.
- Single-page interactive documentation application (`DesignSystem.html` + `designsystem.app.jsx`).
- Comprehensive repository documentation (`PROJECT.md`, `AGENTS.md`, `Architecture.md`, `Design-Principles.md`, `Roadmap.md`, `Contributing.md`).

### Changed
- Rebranded project as standalone, open-source-ready product **Design System Studio**.
- Removed all legacy internal company names and hardcoded references across text, templates, and meta configurations.
