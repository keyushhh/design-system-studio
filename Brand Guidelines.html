Typography

Typeface
Typography is a core part of Design System Studio's visual identity. Our type system is built around clarity, consistency, and modern digital experiences - a defined role for each face that creates hierarchy while staying cohesive across products, marketing, and documentation.
Primary - Space Grotesk
Display, headlines, and large editorial titles. Set tight for a confident, modern voice.
Secondary - Satoshi
The default interface and body typeface - navigation, forms, components, and paragraphs. Chosen for excellent readability on screen.
Tertiary - JetBrains Mono
Reserved for labels, metadata, status, numbers, IDs, and technical information.

Key Elements

Corner Radius

Sharp (0) is the in-product default for an editorial feel; the soft scale (8 / 16px) is reserved for web surfaces.

Shadows

Soft, rare shadows only; the interface leans on spacing and contrast, never heavy elevation.

Patterns & Gradients

A single restrained emerald gradient for cover and divider surfaces; hairline grids for structure. No aggressive multi-stop gradients.

Design System Studio Brand GuidelinesPage 08
Content Section

Layout Grid

A 12-column grid with 16px gutters underpins application layouts. Generous editorial margins and hairline rules separate zones; structure comes from whitespace, not boxes.
