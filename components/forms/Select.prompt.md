Native select, brand-styled - 44px, square, emerald focus ring.

```jsx
<Select label="Template" options={['Business Record','Proposal']} />
<Select label="Region" options={[{value:'apac',label:'APAC'},{value:'emea',label:'EMEA'}]} />
```

`options` take plain strings or `{value,label}`. Sizes sm/md/lg; `error` swaps to the error palette.
