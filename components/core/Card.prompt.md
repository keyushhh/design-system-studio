Flat container surface - hairline border, white or subtle fill, shadow only when raised. The system prefers spacing + contrast over elevation.

```jsx
<Card>Default white card</Card>
<Card surface="subtle" radius="sharp">Warm editorial panel</Card>
<Card elevation="soft" padding={24}>Raised card</Card>
```

`surface`: default / subtle / sunken. `elevation`: flat (default) / soft / lift. `radius`: sharp (in-product) / md / lg (default) / xl.
